Weblinks Categories

Description

Categories help organize your weblinks into logical groups. You can create nested categories (parent-child relationships) to build a hierarchical structure.

Category Management

  • Create categories - Click "New" to add a new category
  • Edit categories - Click on a category title to modify it
  • Nested categories - Set a parent category to create subcategories
  • Category images - Optionally add an image to represent the category
Note: When you delete a category, you can choose what happens to the weblinks inside it (move to another category or trash them).

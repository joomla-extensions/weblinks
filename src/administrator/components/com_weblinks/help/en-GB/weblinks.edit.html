Edit Weblink

Create and manage individual weblink entries in your Joomla directory

Overview

The Edit Weblink screen allows you to add or modify individual weblink entries. This is where you define the link details, URL, categorization, and display options for each weblink in your directory.

💡 Quick Tip: All fields marked with Required must be completed before saving. The weblink will not be visible on the frontend until it is Published.

Main Fields

Title Required
The name or title of the weblink as it will appear on your website. This is the clickable text that users will see.
  • Best practice: Use descriptive titles (e.g., "Joomla Official Website" instead of just "Joomla")
  • Maximum length: 255 characters
  • This field is required and must be unique per category (though duplicates across categories are allowed)
URL Required
The full web address (URL) that users will be directed to when clicking the link.
  • Must include the protocol: http://, https://, or ftp://
  • Example: https://www.joomla.org
  • Important: Always verify that the URL is valid and accessible
  • Relative paths (e.g., /images/logo.png) are also supported for internal links
Alias Optional
The alias is used in SEF (Search Engine Friendly) URLs. If left empty, Joomla will automatically generate one from the title.
  • Only lowercase letters, numbers, and hyphens are allowed (no spaces)
  • Example: joomla-official-website
  • Should be unique to avoid URL conflicts
  • Automatically generated if left blank when saving

Categories & Status

Category Required
Select the category where this weblink will be stored. Categories help organize your weblinks and control where they appear.
  • You can select "Uncategorized" if no category is appropriate
  • To create a new category, click the "+" button next to the field
  • Categories can be nested (parent-child relationships)
  • Categories affect both frontend display and permissions
Status Required
Controls whether the weblink is visible on your website:
  • Published: Visible to visitors with appropriate access levels
  • Unpublished: Hidden from public view (useful for draft content)
  • Archived: Moved to archive, not shown in standard lists
  • Trashed: Soft deleted (can be restored or permanently deleted)

Access & Language

Access Level
Determines which user groups can view this weblink on the frontend.
  • Public: Visible to all visitors (default)
  • Guest: Only visible to non-logged-in users
  • Registered: Only visible to logged-in users
  • Special: Only visible to users with special permissions (e.g., administrators)
  • Custom access levels can be created in Users → Access Levels
Language
Assign the weblink to a specific language (for multilingual sites).
  • All: Shows in all languages (default)
  • Specific language: Only appears when that language is active
  • Requires Joomla's multilingual feature to be enabled
  • Used in combination with language-specific menu items

Metadata & SEO

Description Optional
A brief description of the weblink. This is often displayed in weblink listings and used by search engines.
  • Recommended length: 150-160 characters for SEO
  • Can be used in the frontend layout depending on your template
  • May be displayed as tooltip text on hover
Tags
Add tags to help organize and categorize your weblinks. Tags work across different component types.
  • Tags can be used to group related content across articles, weblinks, and other components
  • Start typing to search for existing tags or create new ones by pressing Enter
  • Tags can be displayed in tag modules and menus
Metadata (Meta Description & Keywords)
Advanced SEO settings that help search engines understand your content.
Meta Description A concise summary of the weblink (155-160 characters maximum). This appears in search engine results.
Meta Keywords Comma-separated keywords relevant to the weblink. Less important for modern SEO but still used by some systems.
Robots Instructions for search engine crawlers (e.g., "index,follow", "noindex,nofollow")
Author Specify an author name for this weblink (overrides global settings)

Images

Link Image Optional
Add an image to represent this weblink. Images can enhance the visual presentation of your weblink directory.
  • Click "Select" to choose an image from the media manager
  • Supports JPEG, PNG, GIF, and SVG formats
  • Recommended size: 100x100 pixels or as defined by your template
  • The image can be displayed in weblink listings based on your menu item settings
Image Position
Control where the image appears in relation to the weblink title and description.
  • Left: Image floats left, text wraps around it
  • Right: Image floats right, text wraps around it
  • Top: Image appears above the content
  • Bottom: Image appears below the content

Advanced Options

Target Window
Specify how the link should open when clicked.
  • Open in parent window: Opens in the same browser tab (default)
  • Open in new window: Opens in a new tab or window (target="_blank")
  • Open in popup: Opens in a popup window (size can be configured)
  • Modal: Opens in a modal dialog (Joomla 4/5 feature)
Link Relationship (rel)
Add HTML rel attributes to the link for SEO and accessibility.
  • nofollow: Tells search engines not to follow this link
  • noopener: Security enhancement for external links opening in new windows
  • noreferrer: Prevents passing referrer information
  • external: Indicates an external link
Multiple attributes can be combined (e.g., "nofollow noopener").

Publishing Information

Created Date The date and time when this weblink was created (automatically set on first save)
Created By The user who created this weblink
Modified Date When this weblink was last updated
Modified By The user who made the last modification
Start Publishing Optional date when the weblink should automatically become published
Finish Publishing Optional date when the weblink should automatically be unpublished
Hits Number of times this weblink has been clicked. Can be reset or manually adjusted.
ID Unique database identifier for this weblink (read-only)
⏰ Scheduling Tip: Use Start/Finish Publishing to automatically manage time-sensitive content like seasonal promotions or event links.

Toolbar Buttons

Save Save the weblink and remain in the edit screen
Save & Close Save the weblink and return to the weblinks list
Save & New Save the current weblink and create a new one
Cancel Discard changes and return to the weblinks list
Help Open this help documentation
Versions View and restore previous versions (if versioning is enabled)
Preview Preview how the weblink will appear on the frontend

Troubleshooting & Best Practices

⚠️ Common Issues:
  • Link not appearing on site: Check that the weblink is Published and the Access Level matches the visitor's permissions
  • Broken links: Always verify URLs before publishing. Consider using a link checker periodically
  • Duplicate titles: While allowed, duplicate titles can confuse users. Try to make titles unique
  • Missing categories: Weblinks without categories may not display in category-based menu items
📌 Best Practices:
  • Use descriptive, unique titles for better SEO
  • Always use HTTPS URLs when available
  • Add meaningful descriptions to help users understand what they'll find
  • Organize weblinks into logical categories
  • Regularly check and update or remove broken links
  • Consider adding images to make your weblink directory more visually appealing

Weblinks Component

Description

The Weblinks component allows you to manage a directory of links on your Joomla site. You can organize links into categories and display them on your site using menu items or modules.

Managing Weblinks

From the main Weblinks screen, you can:

  • Create new weblinks - Click the "New" button in the toolbar
  • Edit existing weblinks - Click on a weblink title or select and click "Edit"
  • Publish/Unpublish - Select weblinks and use the publish/unpublish buttons
  • Trash weblinks - Move unwanted weblinks to the trash
  • Batch process - Use the batch button to move or copy multiple weblinks

Weblink Fields

  • Title - The name of the weblink (required)
  • URL - The full web address including http:// or https:// (required)
  • Category - The category this weblink belongs to
  • Status - Published, unpublished, or trashed
  • Access - Who can view this weblink
  • Language - The language assignment for multilingual sites
Tip: Use categories to organize your weblinks logically. For example, "Resources", "Partners", "Recommended Sites", etc.
